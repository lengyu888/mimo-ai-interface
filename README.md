# 石化厂区全场景北斗/多传感器融合主动感知识别与定位系统

## 项目简介

本项目是基于 MiMo AI 大模型交互系统扩展开发的石化厂区智能感知与定位演示系统，为「石化厂区全场景北斗/多传感器融合主动感知识别与定位技术比赛」而设计。

## 功能特性

### 1. 双页面系统
- **MiMo AI 界面** (`index.html`)：大模型对话交互系统
- **石化厂区感知系统** (`petrochemical.html`)：厂区智能感知与定位系统
- 一键切换功能，无缝切换两种模式

### 2. 石化厂区感知系统功能
#### 厂区地图可视化
- 8个功能区域：炼油一区、催化裂化区、储罐区（高危）、中央控制室、生产装置区、装卸区、维修车间、办公楼
- 建筑物、道路、区域边界可视化
- 小地图导航功能

#### 人员与设备定位
- 实时人员位置追踪（8名在岗人员）
- 设备位置监控（6台关键设备）
- 北斗信标部署（8个信标点）
- 悬浮提示显示详细信息
- 平滑动画效果

#### 传感器监控
- 8种类型的传感器实时数据：
  - 气体探测器（甲烷、CO₂）
  - 温度传感器
  - 压力变送器
  - 液位计
  - 振动传感器
  - 噪声监测
  - 风速风向
  - 可燃气体监测

#### AI 智能分析助手
- 厂区安全状况分析
- 在岗人员分布查询
- 设备状态预测性维护
- 应急预案生成

#### 实时告警系统
- 预警（黄色）与告警（红色）两级显示
- 告警内容与时间记录
- 状态徽章与系统时间显示

#### 数据统计与可视化
- 在岗人员数、设备在线数
- 预警数、告警数
- 定位请求图表（12小时数据）
- 图层切换控制

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **后端**：Python, Flask
- **视觉效果**：SVG, CSS 动画
- **通信协议**：SSE (Server-Sent Events)

## 快速开始

### 环境要求
- Python 3.7+
- 现代浏览器（Chrome, Edge, Firefox）

### 安装与运行

1. **启动服务**
   ```bash
   # Windows 用户直接双击
   start-server.bat
   
   # 或手动运行
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   python server.py
   ```

2. **访问系统**
   - MiMo AI 界面：`http://localhost:8080/`
   - 石化厂区感知：点击顶部「石化厂区感知」按钮切换

## 使用说明

### 页面切换
- 从 MiMo AI 切换到感知系统：点击顶部栏「石化厂区感知」按钮
- 从感知系统切换到 MiMo AI：点击右上角「切换至 MiMo AI」按钮

### 厂区地图交互
- 鼠标悬停查看人员/设备/信标详细信息
- 点击地图元素查看详情
- 使用图层切换按钮控制显示内容

### AI 助手使用
- 点击右侧面板的 AI 建议按钮
- 系统会根据场景提供智能分析

## 比赛项目说明

### 题目编号
XA-202617

### 题目名称
石化厂区全场景北斗/多传感器融合主动感知识别与定位技术比赛

### 研究方向
1. 研究北斗/多源融合定位算法，实现在石化厂区中无缝切换与连续定位
2. 研究遮蔽环境下北斗误差补偿技术，结合传感器多源数据修正定位偏差
3. 研究多传感器联合建模与在线自适应修正技术，提升复杂工况下的定位稳定性
4. 研究针对特定目标（如反光人员、标准设备、仪表状态）实时识别、跟踪与全局定位技术
5. 研制适用于石化厂区全作业场景的多传感器集成定位设备，满足防爆、抗干扰、低功耗、易佩戴等要求，具备低成本、可规模化的推广应用能力
